windows搭建Hugo博客

安装

  • 通过choco
    1.安装Chocolatey

    • 在管理员模式下输入 
      ` @"%SystemRoot%\System32\WindowsPowerShell\v1.0\powershell.exe" -NoProfile -InputFormat None -ExecutionPolicy Bypass -Command "iex ((New-Object System.Net.WebClient).DownloadString('https://chocolatey.org/install.ps1'))" && SET "PATH=%PATH%;%ALLUSERSPROFILE%\chocolatey\bin"  
`

    2.在cmd下输入

    • choco install hugo -confirm
    • choco install hugo -confirm（扩展版，某些主题可能必须）

  • 通过exe文件

    1. 去官网https://github.com/gohugoio/hugo/releases下载相应的文件
    2. 将文件下载好后解压到你需要的文件夹
    3. 在系统中配置环境变量
    4. 在cmd输入hugo version检查是否安装成功

      创建

      1. 用Hugo来创建自己的博客，命令如下图所示：hugo new site myblog（myblog为文件夹的名称）
      2. 通过cd命令进入myblog文件夹，创建themes文件夹，并在Hugo主题商店下载
      3. hugo server -t meme --buildDrafts 来启动nide 博客
      • meme为主题名称
      1. hugo new post\文章名称.md创建你的内容

        部署到GitHub

        123
  1. 建立你的github仓库注意和地址和用户名保持相同
  2. 在myblog目录下执行hugo --theme=m10c --baseUrl="地址" --buildDrafts命令
    • 会自动创建一个public文件夹
    • 然后分别执行git init, git add ., git commot -m + 名字
  3. 执行git remote add origin +地址将其和GitHub进行关联
  4. git push -u origin master推送到你的GitHub上期间注意输入你的用户名密码
  5. 每次更新需重复执行命令2-4
