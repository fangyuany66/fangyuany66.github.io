准备

一.9008回安卓9.0

魅族的底包为9.0所以要先刷回氢os9.0版本

  • 首先解压压缩包
  • 以管理员安装驱动
  • 以管理员打开软件
  • 手机彻底关机，同时按住音量加减，然后插入数据线此时在软件中可以找到你的设备，点击start即可

二.解锁刷twrp及root

工具箱可以打开的直接按工具箱的步骤即可，打不开的按照步骤输入命令行 

注：此为安卓9.0的命令，Q的可能会有一些差距

1.准备

  • 进入开发者选项打开oem解锁，usb调试

  • 打开高级重启

  • 下载及解压工具包

  • cd 命令进入该文件夹下

    2.解锁

  • 重启至fastboot
    电源键选择“引导加载器”重启。
    或者直接输入adb reboot bootloader

  • 在cmd输入fastboot oem unlock

  • 在手机端选择相应的选项

    3.刷入wtrp

  • 在fastboot模式下

  • cmd输入fastboot boot twrp-3.3.1-4-guacamole.img

  • 此时手机会重启到临时TWRP中

  • 将文件夹中的TWRP.zip 和root文件复制进手机卡刷即可

    注：9008之后的系统版本不能直接解锁刷twrp需升级至其余的版本

三.刷入flyme

  • 解压缩刷即包
  • 双击打开.bat文件
  • 按照步骤刷入即可

四.安装xp及主题破解

  • 下载EdXposed Manager

  • 打开后在里面下载以下两个文件

    1.magisk-riru-core
2.magisk-EdXposed

  • 打开面具按照上面的顺序卡刷即可安装成功xp框架

  • 下载安装flyme8助手并在xp框架中勾选即可成功破解主题
