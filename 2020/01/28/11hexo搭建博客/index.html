准备工作

  1. 安装nodejs与npm

    • 打开nojs官网
    • 下载对应系统的版本

  2. 安装cnpm并配置淘宝镜像源

    npm install -g cnpm --registry-https://registry.npm.taobao.org

安装hexo

  1. 通过cnpm install -g hexo-cli命令安装hexo

    初始化hexo

    1. 创建blog文件夹mkdir blog
    2. 初始化sudo hexo init

主题美化（butterfly）

  1. 主题安装
    git clone -b master https://github.com/jerryc127/hexo-theme-butterfly.git themes/Butterfly

  2. 安装所需文件

    • npm install hexo-renderer-pug hexo-renderer-stylus --save or yarn add hexo-renderer-pug hexo-renderer-stylus
    • npm install cheerio@0.22.0 –save

  3. 更改默认主题
    修改_config.yml文件中的theme选项

  4. 配置butterfly主题

    • 在source文件夹下创建_data文件夹
    • 将主题文件夹下的_config文件夹复制到此文件夹下，并更名为butterfly.yml

  5. 修改默认语言
    修改_config文件，默认语言为en需修改为zh-CN

  6. 修改导航栏
    butterfly文件中的menu

    menu:
首页: / || fa fa-home
时间轴: /archives/ || fa fa-archive
标籤: /tags/ || fa fa-tags
分类: /categories/ || fa fa-folder-open
清单||fa fa-heartbeat:
    - 音乐 || /music/ || fa fa-music
    - 照片 || /Gallery/ || fa fa-picture-o
    - 电影 || /movies/ || fa fa-film
友链: /link/ || fa fa-link
关于: /about/ || fa fa-heart，修改

  7. 修改图片
    在butterfly.yml文件中找到avator

  8. 修改网站副标题显示文字

btterfly搜索subtitle，更改其中的文字

绑定到github上

1.新建仓库
￼
2.输入和你gihub名称相同的项目地址
￼

3.修改config文件在文件末尾添加如下内容

deploy:
type: git
repo: https://github.com/fangyuany66/fangyuany66.github.io.git
branch: master

￼

4.执行hexo d命令，输入github的名称及密码即可、
￼
