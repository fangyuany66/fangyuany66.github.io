安装tomcat

安装

  1. 下载tomcat
    官网 https://tomcat.apache.org/download-80.cgi

  2. 解压缩之后直接拷贝到/Users/fy/Documents/Study/java中，压缩后的文件夹名称为tomcat8，位置如下:用户名/资源库

  3. 打开终端输入/Users/fy/Documents/Study/java/tomcat8/bin/startup.sh在3安装成功浏览其中输入localhost：8080出现tomcatjie mian

    配置tomcat

  4. 设置 Tomcat 的环境变量:

    • 输入touch .bash_profile ( pico .bash_profile 也行)

    • 输入命令打开：open -e .bash_profile文稿(nano .bash_profile亦可)

    • 输入Tomcat存储路径export PATH=$PATH:/Users/fy/Documents/Study/java/tomcat8/bin （注意：此处路径是自己保存的 tomcat 存储路径）

    • 更新配置的环境变量，输入命令：source .bash_profile

    • 验证是否成功，终端中输入命令 $PATH

      显示如下-bash: /usr/local/bin:/usr/bin:/bin:/usr/sbin:/sbin:/Users/keso/Library/Tomcat8/bin: No such file or directory

IDEA创建javaweb项目

1.新建项目
　

  1. 在 WEB-INF 目录下点击右键，New -> Directory，创建 classes 和 lib 两个目录

  2. File -> Project Structure，进入 Project Structure窗口，
    点击 Modules -> 选中项目“JavaWeb1” -> 切换到 Paths 选项卡 -> 勾选 “Use module compile output path”，将 “Output path” 和 “Test output path” 都改为之前创建的classes目录

  3. 点击 Modules -> 选中项目“JavaWeb1” -> 切换到 Dependencies 选项卡 -> 点击右边的“+”，选择 “JARs or directories”

  • idea配置完毕点击右上角的三角即可测试是否配置成功
